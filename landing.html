What's Inside

Everything you need, nothing you don't.

Designed for real conversations between real people — secure, fast, and clean.

🔐

Email OTP Verification

Sign up with email + password. A 6-digit code is sent to your inbox via Brevo SMTP — no magic links, no weak links.

💬

Contact-First Messaging

No global chat rooms. Add contacts, get approved, then message privately. Your conversations stay between you and your people.

⚡

Real-Time Delivery

Messages arrive instantly via Supabase Realtime subscriptions. Typing indicators, read receipts, and live presence — all built in.

🤖

AI Chat Assistant

Powered by Groq + Llama 3.3. Get intelligent replies, message summaries, and context-aware assistance right inside your chat.

🛡

Row-Level Security

Every database query is guarded by Supabase RLS policies. You can only read messages you're a participant of — enforced at the DB level.

📎

Rich Messages

Send text, images, files, and reactions. Soft-delete messages, search within conversations, and never lose context.

Getting Started

Up and running in 3 steps.

The smoothest onboarding in private messaging.

1

Sign Up

Enter your email, username, and password. Your account is created instantly.

2

3

Start Chatting

Search for users, send contact requests, and start private conversations.
