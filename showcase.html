✦ Architecture

How It Works

A 4-layer architecture connecting React frontend to real TCP/UDP socket servers through a FastAPI bridge.

01

User Types a Message

The React 18 frontend captures user input with protocol selection (TCP/UDP/AUTO). AI can auto-select the optimal protocol based on message content analysis.

02

WebSocket → FastAPI Backend

Message is sent via WebSocket to the Python FastAPI server. AI Engine screens for toxicity, generates smart reply suggestions, and runs adaptive protocol routing.

03

NetworkBridge → TCP/UDP Servers

The NetworkBridge forwards the message to real TCP (port 9000) or UDP (port 9001) socket servers. Packets are fragmented, checksummed, and tracked with sequence numbers.

04

Delivery & ACK/Drop

TCP guarantees delivery with ACKs and retransmission. UDP fires packets with simulated loss (realistic 5-15% drop rate). Results flow back through WebSocket to update the UI in real-time.
